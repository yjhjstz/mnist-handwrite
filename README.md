### Requirement ###

- Python >=2.7 or >=3.4
  - TensorFlow >=1.7


### How to run ###
* 修改config.py, 配置云预测URL和token.

   - $ pip install -r requirements.txt
   - $ sh build-protoc.sh
   - $ python main.py
