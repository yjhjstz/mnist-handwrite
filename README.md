![screencast](https://cloud.githubusercontent.com/assets/80381/11339453/f04f885e-923c-11e5-8845-33c16978c54d.gif)

### Requirement ###

- Python >=2.7 or >=3.4
  - TensorFlow >=1.7


### How to run ###

    $ pip install -r requirements.txt
    $ sh build-protoc.sh
    $ export host=ip:port
    $ python main.py
